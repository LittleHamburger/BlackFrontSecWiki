有这个想法是在2019年末的时候，当时就想把全流程的渗透测试步骤记录下来，到了2020年3月由于工作原因还没有开始，终于在2020年8月的时候开始一点一点记录渗透测试的每一个环节，当时只是用Markdown开始记录，不知不觉的已经做成了一个大项目。
